# 第 2 章：环境搭建

## 2.1 系统要求

### 操作系统

| 用途 | 推荐系统 | 说明 |
|------|----------|------|
| 推理（生成视频） | Windows 11 / Linux | Windows 需 WSL2 或原生 PowerShell |
| 训练（LoRA 微调） | Linux (Ubuntu 22.04+) | 训练依赖 Triton，仅 Linux 支持 |
| 多 GPU 训练 | Linux | 需要 NCCL + FSDP |

### GPU 要求

- **最低**：NVIDIA GPU + 16GB 显存 + CUDA 12.4+
- **推荐**：NVIDIA GPU + 24GB+ 显存 + CUDA 12.4+
- **训练**：NVIDIA GPU + 80GB 显存（标准）/ 32GB（低显存配置）

### 软件依赖

- Python 3.12+
- CUDA Toolkit 12.4 - 13.x
- Git
- [uv](https://docs.astral.sh/uv/) (Python 包管理器)

## 2.2 安装步骤

### 2.2.1 克隆仓库

```bash
git clone https://github.com/Lightricks/LTX-2.git
cd LTX-2
```

### 2.2.2 安装依赖

```bash
# 安装所有依赖（包括训练和推理）
uv sync --frozen

# 如果只需要推理，可以只安装核心+管道
uv sync --frozen --package ltx-core --package ltx-pipelines

# 安装 xFormers 加速（推荐，非 Blackwell GPU）
uv sync --frozen --extra xformers

# Blackwell GPU (B200) 需要 FlashAttention 4
uv pip install 'flash-attn-4==4.0.0b9'
```

### 2.2.3 激活虚拟环境

```bash
# Linux/Mac
source .venv/bin/activate

# Windows PowerShell
.venv\Scripts\Activate.ps1
```

## 2.3 模型下载

LTX-2 需要下载多个模型文件，按优先级排列：

### 必选模型

#### 1. LTX-2.3 基础模型（二选一）

```bash
# 完整模型（推荐，质量最高）
wget https://huggingface.co/Lightricks/LTX-2.3/resolve/main/ltx-2.3-22b-dev.safetensors

# 蒸馏模型（速度快 5x，质量略降）
wget https://huggingface.co/Lightricks/LTX-2.3/resolve/main/ltx-2.3-22b-distilled-1.1.safetensors
```

| 模型 | 大小 | 推理步数 | 质量 | 速度 |
|------|------|----------|------|------|
| dev (完整) | ~44GB | 40 步 | ⭐⭐⭐⭐⭐ | ⭐⭐ |
| distilled | ~44GB | 8 步 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |

> **对于 AI 动漫**：推荐从 distilled 模型开始，用 ICLoraPipeline 或 DistilledPipeline 快速迭代；最终用完整模型做两阶段高质量渲染。

#### 2. 空间超分模型

```bash
# 2x 超分（两阶段推理必需）
wget https://huggingface.co/Lightricks/LTX-2.3/resolve/main/ltx-2.3-spatial-upscaler-x2-1.1.safetensors

# 1.5x 超分（可选）
wget https://huggingface.co/Lightricks/LTX-2.3/resolve/main/ltx-2.3-spatial-upscaler-x1.5-1.0.safetensors
```

#### 3. 蒸馏 LoRA（两阶段推理必需）

```bash
wget https://huggingface.co/Lightricks/LTX-2.3/resolve/main/ltx-2.3-22b-distilled-lora-384-1.1.safetensors
```

#### 4. Gemma 3 文本编码器

```bash
# 下载整个仓库（约 25GB）
git clone https://huggingface.co/google/gemma-3-12b-it-qat-q4_0-unquantized
# 或者用 huggingface-cli
huggingface-cli download google/gemma-3-12b-it-qat-q4_0-unquantized --local-dir ./models/gemma-3-12b
```

### 动漫相关 LoRA（推荐下载）

```bash
# IC-LoRA Union Control — 综合控制（边缘/深度/姿态）
wget https://huggingface.co/Lightricks/LTX-2.3-22b-IC-LoRA-Union-Control/resolve/main/ltx-2.3-22b-ic-lora-union-control-ref0.5.safetensors

# IC-LoRA Motion Track — 运动轨迹控制
wget https://huggingface.co/Lightricks/LTX-2.3-22b-IC-LoRA-Motion-Track-Control/resolve/main/ltx-2.3-22b-ic-lora-motion-track-control-ref0.5.safetensors

# IC-LoRA Pose Control — 姿态控制
wget https://huggingface.co/Lightricks/LTX-2-19b-IC-LoRA-Pose-Control/resolve/main/ltx-2-19b-ic-lora-pose-control.safetensors

# IC-LoRA Detailer — 细节增强
wget https://huggingface.co/Lightricks/LTX-2-19b-IC-LoRA-Detailer/resolve/main/ltx-2-19b-ic-lora-detailer.safetensors

# 摄影机运动 LoRA（可选）
wget https://huggingface.co/Lightricks/LTX-2-19b-LoRA-Camera-Control-Dolly-In/resolve/main/ltx-2-19b-lora-camera-control-dolly-in.safetensors
wget https://huggingface.co/Lightricks/LTX-2-19b-LoRA-Camera-Control-Dolly-Out/resolve/main/ltx-2-19b-lora-camera-control-dolly-out.safetensors
wget https://huggingface.co/Lightricks/LTX-2-19b-LoRA-Camera-Control-Static/resolve/main/ltx-2-19b-lora-camera-control-static.safetensors
```

### 推荐目录结构

```
LTX-2/
├── models/
│   ├── ltx-2.3-22b-dev.safetensors           # 完整模型 (~44GB)
│   ├── ltx-2.3-22b-distilled-1.1.safetensors # 蒸馏模型 (~44GB)
│   ├── ltx-2.3-spatial-upscaler-x2-1.1.safetensors  # 空间超分
│   ├── ltx-2.3-22b-distilled-lora-384-1.1.safetensors # 蒸馏LoRA
│   ├── gemma-3-12b/                           # 文本编码器 (~25GB)
│   └── loras/                                  # 各种 LoRA
│       ├── ic-lora-union-control.safetensors
│       ├── ic-lora-pose-control.safetensors
│       └── ...
├── outputs/                                    # 生成结果
└── projects/                                   # 项目文件
```

### 生成测试视频（蒸馏模型，最快）

```bash
python -m ltx_pipelines.distilled \
    --checkpoint-path models/ltx-2.3-22b-distilled-1.1.safetensors \
    --spatial-upsampler-path models/ltx-2.3-spatial-upscaler-x2-1.1.safetensors \
    --gemma-root models/gemma-3-12b \
    --prompt "An anime style girl with flowing hair standing in a cherry blossom field, wind blowing petals, soft lighting" \
    --output-path outputs/test_anime.mp4 \
    --height 512 \
    --width 768 \
    --num-frames 49
```

> 预期：在 RTX 4090 上约 2-3 分钟生成 49 帧（约 2 秒 @24fps）视频。

## 2.6 常见安装问题

### CUDA 版本不匹配

```bash
# 检查 CUDA 版本
nvidia-smi  # 查看驱动支持的最高 CUDA 版本
python -c "import torch; print(torch.version.cuda)"  # PyTorch 编译的 CUDA 版本

# 如果不匹配，重装 PyTorch
uv pip install torch --index-url https://download.pytorch.org/whl/cu124
```

### 显存不足

```bash
# 使用 FP8 量化
PYTORCH_CUDA_ALLOC_CONF=expandable_segments:True \
python -m ltx_pipelines.ti2vid_two_stages \
    --quantization fp8-cast \
    ... # 其他参数

# 使用 Block Streaming（最省显存）
# 在 Python API 中设置 offload_mode=OffloadMode.CPU 或 OffloadMode.DISK
```

### Windows 特殊注意事项

```powershell
# PowerShell 设置环境变量
$env:PYTORCH_CUDA_ALLOC_CONF = "expandable_segments:True"

# 路径使用正斜杠或转义反斜杠
python -m ltx_pipelines.distilled `
    --checkpoint-path "G:/AI/LTX-2/models/ltx-2.3-22b-distilled-1.1.safetensors" `
    ...
```
