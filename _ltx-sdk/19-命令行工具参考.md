# 第 19 章：命令行工具参考

> 所有 CLI 脚本用法速查

## 19.1 数据集处理

### split_scenes.py — 场景检测与切割

```bash
python scripts/split_scenes.py \
    --input_dir raw_videos/ \
    --output_dir scenes/ \
    --min_duration 2.0 \
    --max_duration 10.0 \
    --threshold 30.0
```

| 选项 | 说明 |
|------|------|
| `--input_dir` | 原始视频目录 |
| `--output_dir` | 输出场景片段目录 |
| `--min_duration` | 最小场景时长 (秒) |
| `--max_duration` | 最大场景时长 (秒) |
| `--threshold` | 场景切换检测阈值 |

### caption_videos.py — 视频字幕生成

```bash
python scripts/caption_videos.py \
    --input_dir scenes/ \
    --output_dir captions/ \
    --backend qwen3-omni
```

| 后端 | 质量 | 速度 | 要求 |
|------|------|------|------|
| Qwen3-Omni | 最高 | 慢 | GPU |
| Gemini | 良好 | 快 | API key |

### process_dataset.py — 潜在编码 + 嵌入预计算

```bash
python scripts/process_dataset.py \
    --dataset_dir scenes/ \
    --captions_dir captions/ \
    --output_dir precomputed/ \
    --model_path checkpoint.safetensors \
    --text_encoder_path gemma-3-12b \
    --num_gpus 4 \
    --resolution_buckets "480p,720p,1080p"
```

### process_videos.py — 仅视频潜在编码

```bash
python scripts/process_videos.py \
    --input_dir scenes/ \
    --output_dir latents/ \
    --model_path checkpoint.safetensors
```

### process_captions.py — 仅文本嵌入

```bash
python scripts/process_captions.py \
    --captions_dir captions/ \
    --output_dir embeddings/ \
    --text_encoder_path gemma-3-12b
```

## 19.2 参考与调试

### compute_reference.py — IC-LoRA 参考生成

```bash
python scripts/compute_reference.py \
    --input_dir scenes/ \
    --output_dir references/ \
    --model_path checkpoint.safetensors
```

### decode_latents.py — 潜在解码调试

```bash
python scripts/decode_latents.py \
    --latents_dir precomputed/video_latents/ \
    --output_dir decoded/ \
    --model_path checkpoint.safetensors
```

将预计算的潜在解码为可播放视频，用于验证预处理质量。

## 19.3 训练

### train.py — 主训练脚本

```bash
# 基本用法
python scripts/train.py configs/t2v_lora.yaml

# 覆盖配置参数
python scripts/train.py configs/t2v_lora.yaml \
    --model.model_path /custom/path.safetensors \
    --optimization.max_steps 5000

# 从检查点恢复
python scripts/train.py configs/t2v_lora.yaml \
    --resume_from output_dir/checkpoints/step-1000
```
