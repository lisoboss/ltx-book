# 前言

## 关于本书

本书是 **LTX-2 SDK 文档**，面向需要深入理解 LTX-2 Python SDK 的开发者、研究人员和技术用户。

LTX-2 是首个基于 DiT（Diffusion Transformer）架构的音视频基础模型，将现代视频生成的所有核心能力统一在一个模型中。其 Python SDK 由三个核心包组成：

| 包 | 用途 | 核心内容 |
|------|------|----------|
| **ltx-core** | 模型核心 | Transformer、VAE、Vocoder、文本编码器、调度器、引导器 |
| **ltx-pipelines** | 推理管道 | 11 种生产级推理管道，从文本/图像/音频生成视频 |
| **ltx-trainer** | 训练工具 | LoRA 微调、全量微调、IC-LoRA、14 种训练模式 |

## 本书结构

本书按模块分为四个部分：

### 第一部分：入门（第 1 章）
概述、环境搭建、模型下载，让你快速上手。

### 第二部分：ltx-core — 模型核心（第 2–7 章）
深入 LTX-2 的架构细节：非对称双流 Transformer、VAE 与 Vocoder、Gemma 文本编码器、扩散组件、加载与量化。

### 第三部分：ltx-pipelines — 推理（第 8–12 章）
11 种推理管道的选择与使用、多模态引导机制、性能优化策略。

### 第四部分：ltx-trainer — 训练（第 13–20 章）
灵活策略框架、数据集准备、14 种训练模式、配置参考、分布式训练、自定义策略、故障排除。
