# 第 15 章：训练模式

> 14 种训练模式详解：T2V、I2V、扩展、IC-LoRA、修复、A2V、V2A、AV2AV 等

## 15.1 模式快速参考

FlexibleStrategy 通过组合 `is_generated` 和 `conditions` 表达所有模式：

| 模式 | 配置 | 条件类型 |
|------|------|----------|
| T2V | video: generated, audio: none | 无（纯文本） |
| T2A | video: none, audio: generated | 无（纯文本） |
| I2V | video: generated, audio: none | first_frame |
| Video Extend (prefix) | video: generated | prefix |
| Video Extend (suffix) | video: generated | suffix |
| V2V IC-LoRA | video: generated | reference |
| Video Inpainting | video: generated | mask |
| Video Outpainting | video: generated | spatial_crop |
| A2V | video: generated, audio: frozen | video_to_audio |
| V2A | video: frozen, audio: generated | audio_to_video |
| A2A IC-LoRA | audio: generated | reference (audio) |
| AV2AV IC-LoRA | video: generated, audio: generated | reference (video+audio) |
| Audio Extend (prefix) | audio: generated | prefix (audio) |
| Audio Inpainting | audio: generated | mask (audio) |

## 15.2 详细配置示例

### 文本到视频 (T2V)

```yaml
training_strategy:
  name: flexible
  video:
    is_generated: true
    latents_dir: latents/video
    conditions: []
  audio:
    is_generated: false
```

### 图像到视频 (I2V)

```yaml
video:
  is_generated: true
  latents_dir: latents/video
  conditions:
    - type: first_frame
      latents_dir: latents/first_frame
```

### 视频扩展 (Extension)

```yaml
# 前向扩展
video:
  conditions:
    - type: prefix
      latents_dir: latents/prefix
      num_frames: 1

# 后向扩展
video:
  conditions:
    - type: suffix
      latents_dir: latents/suffix
      num_frames: 1
```

### IC-LoRA 视频到视频

```yaml
video:
  is_generated: true
  latents_dir: latents/video
  conditions:
    - type: reference
      latents_dir: latents/reference
```

### 视频修复 (Inpainting)

```yaml
video:
  is_generated: true
  conditions:
    - type: mask
      latents_dir: latents/mask
```

### 音频到视频 + 视频到音频

```yaml
# A2V: 音频冻结，视频生成
video:
  is_generated: true
  conditions:
    - type: video_to_audio
      latents_dir: latents/audio
audio:
  is_generated: false  # frozen

# V2A: 视频冻结，音频生成
video:
  is_generated: false  # frozen
audio:
  is_generated: true
  conditions:
    - type: audio_to_video
      latents_dir: latents/video
```

### AV2AV IC-LoRA

```yaml
video:
  is_generated: true
  conditions:
    - type: reference
      latents_dir: latents/video_reference
audio:
  is_generated: true
  conditions:
    - type: reference
      latents_dir: latents/audio_reference
```

## 15.3 模式与推断管道映射

| 训练模式 | 对应的推断管道 |
|----------|---------------|
| T2V | TI2VidTwoStagesPipeline |
| I2V | TI2VidTwoStagesPipeline (首帧条件) |
| V2V IC-LoRA | ICLoraPipeline |
| Video Extend | TI2VidTwoStagesPipeline (prefix condition) |
| A2V | A2VidPipelineTwoStage |
| LipDub | LipDubPipeline |

## 15.4 从旧版策略迁移

`TextToVideoStrategy` 和 `VideoToVideoStrategy` 已弃用。迁移方法：

```yaml
# 旧版
training_strategy:
  name: text_to_video
  video_latents_dir: latents/video

# 新版 FlexibleStrategy
training_strategy:
  name: flexible
  video:
    is_generated: true
    latents_dir: latents/video
    conditions: []
```
