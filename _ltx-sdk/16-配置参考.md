# 第 16 章：配置参考

> Pydantic 配置系统完整参考：ModelConfig、LoraConfig、OptimizationConfig 等

## 16.1 配置架构

所有配置定义在 `src/ltx_trainer/config.py` 中，使用 Pydantic 验证，`extra="forbid"`（未知字段会报错）。

### 顶层配置

```yaml
# LtxTrainerConfig
model:         # ModelConfig — 模型路径和训练模式
training:      # TrainingStrategyConfig — 灵活策略
optimization:  # OptimizationConfig — 学习率、批次等
acceleration:  # AccelerationConfig — 混合精度、量化
data:          # DataConfig — 数据路径
validation:    # ValidationConfig — 验证样本
checkpoints:   # CheckpointsConfig — 保存策略
flow_matching: # FlowMatchingConfig — 时间步采样
hub:           # HubConfig — HuggingFace Hub 推送
wandb:         # WandbConfig — W&B 日志
```

## 16.2 ModelConfig

```yaml
model:
  model_path: /path/to/checkpoint.safetensors
  text_encoder_path: /path/to/gemma-3-12b
  training_mode: lora          # lora | full
  lora:
    rank: 32                   # LoRA 秩
    alpha: 16                  # 缩放因子
    target_modules: [qkv, ff]  # 目标模块列表
    dropout: 0.0
```

## 16.3 OptimizationConfig

```yaml
optimization:
  learning_rate: 1e-4
  batch_size: 1
  gradient_accumulation_steps: 4   # 等效 batch = 4
  max_steps: 10000
  lr_scheduler: cosine
  lr_warmup_steps: 500
  enable_gradient_checkpointing: true
```

## 16.4 AccelerationConfig

```yaml
acceleration:
  mixed_precision: bf16        # bf16 | fp16 | no
  quantization: null           # null | fp8-cast | fp8-scaled-mm
  num_processes: 1             # GPU 数量
  use_8bit_text_encoder: false # 8-bit Gemma (bitsandbytes)
  gradient_checkpointing: true
```

## 16.5 ValidationConfig

```yaml
validation:
  prompt: "A beautiful sunset over the ocean"
  width: 768
  height: 512
  frames: 49
  seed: 42
  cfg_scale: 3.0
  stg_scale: 1.0
  num_inference_steps: 30
  save_every_n_steps: 500
  conditions: []  # 验证条件列表
```

### 验证条件类型

```yaml
conditions:
  - type: first_frame
    image_path: /path/to/image.jpg
  - type: prefix
    video_path: /path/to/prefix.mp4
  - type: reference
    video_path: /path/to/reference.mp4
```

## 16.5 CheckpointsConfig

```yaml
checkpoints:
  save_every_n_steps: 1000
  max_keep: 5              # 保留最近 N 个检查点
  save_fp8: false           # 同时保存 FP8 版本
```

## 16.6 FlowMatchingConfig

```yaml
flow_matching:
  timestep_sampler: shifted-logit-normal
  min_timestep: 0.0
  max_timestep: 1.0
  uniform_fraction: 0.1     # 10% 均匀采样（防分布塌缩）
```
