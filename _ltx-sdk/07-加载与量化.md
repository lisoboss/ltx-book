# 第 7 章：加载与量化

> SingleGPUModelBuilder、LoRA 融合、FP8 量化后端、块流式处理

## 7.1 SingleGPUModelBuilder

`SingleGPUModelBuilder` 是 ltx-core 的统一模型加载器，支持从 safetensors checkpoint 加载所有模型组件：

```python
from ltx_core.loader.single_gpu_model_builder import SingleGPUModelBuilder
from ltx_core.loader.sft_loader import SafetensorsModelStateDictLoader
from ltx_core.loader.sd_ops import SDOps

builder = SingleGPUModelBuilder(
    checkpoint_path="ltx-2.3-22b-dev.safetensors",
    device="cuda",
    dtype=torch.bfloat16,
)
```

### LoRA 融合

`SingleGPUModelBuilder` 支持在加载时融合 LoRA 权重：

```python
builder = SingleGPUModelBuilder(
    checkpoint_path="base_model.safetensors",
    lora_path="my_lora.safetensors",
    lora_scale=1.0,
)
```

LoRA 权重在加载时与基础模型权重合并（fuse），无需运行时额外计算。

## 7.2 组件级加载器

除了统一的 `SingleGPUModelBuilder`，ltx-trainer 也提供组件级加载器：

```python
from ltx_trainer.model_loader import (
    load_transformer,       # → LTXModel
    load_video_vae_encoder, # → VideoEncoder
    load_video_vae_decoder, # → VideoDecoder
    load_audio_vae_decoder, # → AudioDecoder
    load_vocoder,           # → Vocoder or VocoderWithBWE (auto)
    load_text_encoder,      # → GemmaTextEncoder (Gemma LLM only)
    load_embeddings_processor, # → EmbeddingsProcessor (feature extractor + connectors)
    load_model,             # → LtxModelComponents (convenience wrapper)
)
```

## 7.3 量化 (Quantization)

LTX-2 支持两种 FP8 量化后端：

### FP8 Cast

```python
# CLI
--quantization fp8-cast

# Python
from ltx_core.quantization import QuantizationPolicy
builder = SingleGPUModelBuilder(..., quantization=QuantizationPolicy.fp8_cast())
```

- 将 bf16 checkpoint 即时转换为 FP8
- **适用于 bf16 checkpoint**
- 降低内存占用

### FP8 Scaled MM (Hopper GPU)

```python
# CLI
--quantization fp8-scaled-mm

# Python
builder = SingleGPUModelBuilder(..., quantization=QuantizationPolicy.fp8_scaled_mm())
```

- 使用 FP8 缩放矩阵乘法
- **适用于 fp8 checkpoint**
- 需要 Hopper GPU + TensorRT-LLM
