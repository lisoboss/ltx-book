# 第 6 章：扩散组件

> 调度器、引导器（CFG/STG/APG）、加噪器、分块器

## 6.1 调度器 (Scheduler)

`LTX2Scheduler` 管理去噪过程中的 sigma 时间表：

```python
from ltx_core.components.schedulers import LTX2Scheduler

scheduler = LTX2Scheduler()
# 默认步数: 30 (LTX-2.3) / 40 (LTX-2)
sigmas = scheduler.execute(num_steps=30)
```

不同管道的步数：
- **两阶段（完整模型）**：30-40 步
- **HQ 管道**：15 步（Res2s 二阶采样器 + token 计数相关的 sigma shift）
- **蒸馏管道**：固定 8 步（`DISTILLED_SIGMA_VALUES`），Stage 2 用 3 步（`STAGE_2_DISTILLED_SIGMA_VALUES`）

## 6.2 引导器 (Guiders)

引导器实现不同的无分类器引导策略：

| 引导器 | 说明 | 参数 |
|--------|------|------|
| **CFGGuider** | 经典无分类器引导 | `cfg_scale=3.0` (video), `7.0` (audio) |
| **STGGuider** | 时空引导 — 扰动自注意力 | `stg_scale=1.0`, `stg_blocks=[28]` 或 `[29]` |
| **Modality Guider** | 跨模态注意力缩放 | `modality_scale=3.0` |

### CFG (Classifier-Free Guidance)

混合条件预测和非条件预测：

```python
from ltx_core.components.guiders import CFGGuider

guider = CFGGuider(cfg_scale=3.0)
# 混合: result = uncond + scale * (cond - uncond)
```

### STG (Spatio-Temporal Guidance)

在指定 Transformer 块中扰动自注意力：

```python
guider = STGGuider(stg_scale=1.0, stg_blocks=[28])
# stg_blocks: LTX-2.3 默认 [28], LTX-2 默认 [29]
```

### Modality Guidance

缩放视频和音频流之间的跨模态注意力：

```python
guider = ModalityGuider(modality_scale=3.0)
```

> 注意：所有引导仅在 Stage 1 生效。Stage 2 始终使用 `SimpleDenoiser`（无引导）。

## 6.3 加噪器 (Noiser)

加噪器负责向潜在特征添加噪声，实现前向扩散过程：

```python
from ltx_core.components.noisers import Noiser

noiser = Noiser(scheduler)
noisy_latent = noiser.add_noise(clean_latent, sigma, noise)
```

## 6.4 分块器 (Patchifier)

将 VAE 潜在表示转换为 Transformer 期望的 token 序列：

- `VideoLatentPatchifier`：`[B, 128, F, H, W]` → `[B, seq_len, 128]`
- `AudioPatchifier`：`[B, 8, mel_frames, 16]` → `[B, seq_len, 128]`
