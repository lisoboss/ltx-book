# 第 1 章：概述与环境搭建

> LTX-2 SDK 是什么、仓库结构、安装与模型下载

## 1.1 LTX-2 简介

LTX-2 是首个基于 DiT（Diffusion Transformer）架构的**音视频基础模型**，由 Lightricks 开源。它将现代视频生成的所有核心能力统一在一个模型中：

- **同步音视频生成**：同时输出视频和音频
- **高保真度**：生产级输出质量
- **多种性能模式**：从快速原型到最高质量
- **API 访问 + 开源**：既可通过 ltx.video 使用，也可本地部署

LTX-2.3 是当前推荐版本（22B 参数），相比初代 LTX-2（19B）在文本条件控制和音频质量上有显著提升。

## 1.2 仓库结构

LTX-2 采用 Python monorepo 结构，使用 `uv` 管理依赖：

```
LTX-2/
├── packages/
│   ├── ltx-core/          # 核心模型实现
│   │   └── src/ltx_core/
│   │       ├── model/transformer/   # DiT 模型
│   │       ├── model/video_vae/     # Video VAE 编解码器
│   │       ├── model/audio_vae/     # Audio VAE + Vocoder
│   │       ├── text_encoders/gemma/ # Gemma 文本编码器
│   │       ├── components/          # 调度器、引导器、分块器
│   │       ├── conditioning/        # 条件控制
│   │       └── loader/              # 模型加载器
│   ├── ltx-pipelines/      # 推理管道
│   │   └── src/ltx_pipelines/  # 11 种管道实现
│   └── ltx-trainer/        # 训练工具
│       ├── src/ltx_trainer/    # 训练核心模块
│       ├── scripts/            # CLI 脚本
│       ├── configs/            # 示例配置（18 个 yaml）
│       └── docs/               # 7 个详细指南文档
├── pyproject.toml
└── README.md
```

## 1.3 系统要求

| 用途 | 要求 |
|------|------|
| 操作系统 | Linux（训练必须）；Windows/WSL（推理支持） |
| Python | 3.12+ |
| CUDA | 12.4+ |
| GPU 推理 | 24GB+ VRAM 推荐（16GB 最低） |
| GPU 训练 | 80GB+ VRAM 推荐（32GB 最低 VRAM 配置） |
| 包管理器 | [uv](https://docs.astral.sh/uv/) |

## 1.4 安装步骤

```bash
# 克隆仓库
git clone https://github.com/Lightricks/LTX-2.git
cd LTX-2

# 安装所有依赖（含训练和推理）
uv sync --frozen

# 如果只需要推理
uv sync --frozen --package ltx-core --package ltx-pipelines

# 安装 xFormers 加速（推荐，非 Blackwell GPU）
uv sync --frozen --extra xformers

# Blackwell GPU (B200) 需要 FlashAttention 4
uv pip install 'flash-attn-4==4.0.0b9'

# 激活虚拟环境
source .venv/bin/activate
```

## 1.5 模型下载

从 [LTX-2.3 HuggingFace](https://huggingface.co/Lightricks/LTX-2.3) 下载：

### 必需模型

| 模型 | 文件 | 大小 | 说明 |
|------|------|------|------|
| 基础模型 | `ltx-2.3-22b-dev.safetensors` | ~44GB | 完整模型，质量最高 |
| 蒸馏模型 | `ltx-2.3-22b-distilled-1.1.safetensors` | ~44GB | 8 步推理，速度快 5x |
| 空间上采样器 | `ltx-2.3-spatial-upscaler-x2-1.1.safetensors` | — | 两阶段管道必需 |
| 蒸馏 LoRA | `ltx-2.3-22b-distilled-lora-384-1.1.safetensors` | — | 两阶段管道必需（蒸馏管道除外） |
| 文本编码器 | [Gemma 3 12B](https://huggingface.co/google/gemma-3-12b-it-qat-q4_0-unquantized) | — | 下载整个仓库 |

### 可选 LoRA

| LoRA | 用途 |
|------|------|
| IC-LoRA Union Control | 通用图像条件控制 |
| IC-LoRA Motion Track | 运动轨迹控制 |
| IC-LoRA Detailer | 细节增强 |
| IC-LoRA Pose Control | 姿态控制 |
| Camera Control 系列 | 镜头运动控制（推拉摇移等 7 种） |
| IC-LoRA HDR | HDR 输出 |
| IC-LoRA LipDub | 唇形同步 |

### 空间上采样器

- `ltx-2.3-spatial-upscaler-x2-1.1.safetensors` — 2x 空间上采样
- `ltx-2.3-spatial-upscaler-x1.5-1.0.safetensors` — 1.5x 空间上采样
