# 第 9 章：核心推理管道

> TI2VidTwoStagesPipeline、TI2VidTwoStagesHQPipeline、TI2VidOneStagePipeline

## 9.1 TI2VidTwoStagesPipeline（推荐默认）

生产质量的文本/图像到视频生成。这是最常用的管道。

### 架构

```
Stage 1 (低分辨率)                Stage 2 (高分辨率)
┌──────────────────┐              ┌──────────────────┐
│ Full Model       │              │ Distilled LoRA   │
│ + CFG/STG 引导    │              │ + SimpleDenoiser │
│ + Euler 采样      │              │ + Euler 采样      │
│ 30-40 步         │              │ 8 步             │
│ 输出: 低分辨率视频 │ ── 2x Upsample ──→ │ 输出: 最终视频     │
└──────────────────┘              └──────────────────┘
```

### 条件支持

- **文本条件**：通过 prompt + Gemma 文本编码器
- **图像条件**：首帧（`frame_idx=0` 替换 latent）+ 关键帧（`frame_idx>0` 引导）

### CLI 使用

```bash
python -m ltx_pipelines.ti2vid_two_stages     --model_path ltx-2.3-22b-dev.safetensors     --distilled_lora_path ltx-2.3-22b-distilled-lora-384-1.1.safetensors     --spatial_upscaler_path ltx-2.3-spatial-upscaler-x2-1.1.safetensors     --text_encoder_path gemma-3-12b     --prompt "A cat running across a field"     --output_path output.mp4     --width 768 --height 512 --frames 49
```

### Python API

```python
from ltx_pipelines.ti2vid_two_stages import TI2VidTwoStagesPipeline

pipeline = TI2VidTwoStagesPipeline(
    model_path="ltx-2.3-22b-dev.safetensors",
    text_encoder_path="gemma-3-12b",
    distilled_lora_path="ltx-2.3-22b-distilled-lora-384-1.1.safetensors",
    spatial_upscaler_path="ltx-2.3-spatial-upscaler-x2-1.1.safetensors",
)

video = pipeline(
    prompt="A cat running across a field",
    width=768, height=512, frames=49,
    cfg_scale=3.0, stg_scale=1.0,
)
```

## 9.2 TI2VidTwoStagesHQPipeline（最高质量）

与 TwoStages 架构相同，但使用 **Res2s 二阶采样器**：

- 更少的步数（15 步 vs 30-40 步）
- **更好的质量**（二阶方法收敛更好）
- 蒸馏 LoRA 应用于**两个阶段**（而非仅 Stage 2）
- 独立的 `distilled_lora_strength_stage_1` / `_stage_2` 参数
- STG 禁用（`stg_scale=0.0`）
- Sigma 调度依赖 token 计数（`latent=` 参数传递给 scheduler）

## 9.3 TI2VidOneStagePipeline（快速原型）

单阶段生成，用于快速测试：

- 仅 Stage 1，无空间上采样
- 使用完整模型 + Euler 采样
- 输出分辨率较低
- 适合快速迭代 prompt 和参数

```bash
python -m ltx_pipelines.ti2vid_one_stage     --model_path ltx-2.3-22b-dev.safetensors     --text_encoder_path gemma-3-12b     --prompt "test prompt"     --output_path test.mp4
```
