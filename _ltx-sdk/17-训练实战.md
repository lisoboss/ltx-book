# 第 17 章：训练实战

> 单GPU/多GPU 训练、DDP/FSDP、HuggingFace Hub、W&B、自动调优

## 17.1 单 GPU 训练

```bash
cd packages/ltx-trainer
uv run python scripts/train.py configs/t2v_lora.yaml
```

输出目录结构：

```
output_dir/
├── checkpoints/
│   ├── step-1000/
│   │   ├── lora_weights.safetensors
│   │   └── config.yaml
│   └── step-2000/
├── validation/
│   ├── step-0500.mp4
│   └── step-1000.mp4
├── logs/
│   └── training.log
└── wandb/
```

## 17.2 多 GPU / 分布式训练

### DDP (Distributed Data Parallel)

```bash
uv run accelerate launch \
    --num_processes 4 \
    scripts/train.py configs/t2v_lora.yaml
```

### FSDP (Fully Sharded Data Parallel)

用于全量微调的大模型训练：

```yaml
# configs/accelerate/fsdp.yaml
compute_environment: LOCAL_MACHINE
distributed_type: FSDP
fsdp_config:
  fsdp_auto_wrap_policy: TRANSFORMER_BASED_WRAP
  fsdp_transformer_layer_cls_to_wrap: BasicAVTransformerBlock
```

```bash
uv run accelerate launch \
    --config_file configs/accelerate/fsdp.yaml \
    scripts/train.py configs/t2v_full.yaml
```

## 17.5 自动调优

训练器根据硬件自动调整参数：VRAM 检测、Batch size 调整、梯度累积、混合精度推荐。

## 17.6 低显存配置 (32GB)

```yaml
optimization:
  batch_size: 1
  gradient_accumulation_steps: 8
  enable_gradient_checkpointing: true
acceleration:
  mixed_precision: bf16
  use_8bit_text_encoder: true
model:
  lora:
    rank: 16
    target_modules: [qkv]
```
