# 第 5 章：文本编码器

> Gemma 3 三块流水线、特征提取器 V1 vs V2、嵌入连接器

## 5.1 三块流水线

`GemmaTextEncoder` 实现了一个三块流水线，将文本 prompt 转换为 Transformer 可用的 embeddings：

```
输入: "一只猫在草地上奔跑"
      │
      ▼
┌───────────────────┐
│ Block 1: Gemma LLM │  Tokenization → Gemma 3 12B 前向传播
│                    │  输出: hidden_states
└────────┬──────────┘
         │
         ▼
┌──────────────────────┐
│ Block 2:             │
│ Feature Extractor    │  hidden_states → 归一化特征
│ (V1 or V2)           │  输出: features
└────────┬─────────────┘
         │
         ▼
┌──────────────────────┐
│ Block 3:             │
│ Embeddings Processor │  features → embeddings connectors
│ (Connectors)         │  输出: video_embeds [4096-dim]
│                      │        audio_embeds [2048-dim]
└──────────────────────┘
```

## 5.2 Block 1: Gemma LLM

- 模型：Gemma 3 12B（QAT Q4_0 量化版本）
- 输入：文本 prompt，经过 tokenizer
- 输出：最后一层 hidden states
- 路径：加载自 `google/gemma-3-12b-it-qat-q4_0-unquantized`

## 5.3 Block 2: Feature Extractor

两个版本，自动检测：

| 特征 | FeatureExtractorV1 (19B) | FeatureExtractorV2 (22B) |
|------|--------------------------|--------------------------|
| 输出 | 单一的 `aggregate_embed` | 分离的 `video_aggregate_embed` + `audio_aggregate_embed` |
| 归一化 | 基础 | 逐 token RMSNorm |
| Caption Proj | 在 Transformer 中 | 在 Feature Extractor 中 |
| 视频/音频差异 | 相同输出 | 独立处理 |

检测逻辑在 `_create_feature_extractor()` 中：检查 V2 配置键（`caption_proj_before_connector` 等），存在则 V2，否则 V1。

## 5.4 Block 3: Embeddings Processor

连接器（Connectors）将特征投影到 Transformer 期望的维度：

- **视频连接器**：输出 4096-dim embeddings
- **音频连接器**：输出 2048-dim embeddings
- V2 中音频连接器使用独立的 `AudioEmbeddings1DConnectorConfigurator`（V1 回退到视频连接器键）

### 预计算 Embeddings

训练时，Block 1+2 可以离线预计算（`process_captions.py`），Block 3 在训练过程中实时应用：

```python
# 离线预计算 (process_captions.py)
video_features, audio_features, attention_mask = text_encoder.precompute(
    prompt, padding_side="left"
)

# 训练时实时应用连接器 (Block 3 only)
video_embeds, audio_embeds, binary_mask = text_encoder.embeddings_processor.create_embeddings(
    video_features, audio_features, additive_mask
)
```

预计算后，训练器会卸载 Gemma LLM 和 Feature Extractor 以释放 VRAM，只保留 Embeddings Processor（连接器）。
