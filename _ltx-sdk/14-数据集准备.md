# 第 14 章：数据集准备

> 场景分割、视频字幕、预处理流水线、分辨率桶、输出结构

## 14.1 预处理流水线

```
原始视频
   │
   ▼
┌──────────────────┐
│ 1. split_scenes  │  场景检测与切割
└────────┬─────────┘
         │ 场景片段
         ▼
┌──────────────────┐
│ 2. caption_videos│  自动字幕生成 (Qwen3-Omni / Gemini)
└────────┬─────────┘
         │ 视频 + 字幕
         ▼
┌──────────────────┐
│ 3. process_dataset│ VAE 潜在编码 + 文本嵌入
│                  │  (多 GPU 支持)
└────────┬─────────┘
         │ 潜在 + 嵌入 (训练就绪)
         ▼
      训练
```

## 14.2 split_scenes.py

场景检测和切割：

```bash
python scripts/split_scenes.py     --input_dir raw_videos/     --output_dir scenes/     --min_duration 2.0     --max_duration 10.0
```

## 14.3 caption_videos.py

自动视频字幕生成，支持多个后端：

```bash
# Qwen3-Omni (推荐，质量最高)
python scripts/caption_videos.py     --input_dir scenes/     --output_dir captions/     --backend qwen3-omni

# Gemini
python scripts/caption_videos.py     --input_dir scenes/     --output_dir captions/     --backend gemini
```

## 14.4 process_dataset.py

VAE 潜在编码 + 文本嵌入预计算：

```bash
# 单 GPU
python scripts/process_dataset.py     --dataset_dir scenes/     --captions_dir captions/     --output_dir precomputed/     --model_path checkpoint.safetensors     --text_encoder_path gemma-3-12b

# 多 GPU（自动分片）
python scripts/process_dataset.py     --dataset_dir scenes/     --num_gpus 4     ...
```

### 分辨率桶（Resolution Buckets）

预处理自动按分辨率分组：

| 桶 | 尺寸 |
|-----|------|
| 480p | 768 × 480 |
| 720p | 1280 × 720 |
| 1080p | 1920 × 1080 |

### 数据格式

- **潜在**：`[C, F, H, W]` 格式（新格式）
- **嵌入**：`video_prompt_embeds` + `audio_prompt_embeds`（新格式）
- 自动处理旧格式（`prompt_embeds`）向后兼容
- 旧格式 `[seq_len, C]` 自动逆分块化为 `[C, F, H, W]`

## 14.5 特殊数据准备

### IC-LoRA 参考

```bash
python scripts/compute_reference.py     --input_dir scenes/     --output_dir references/
```

### 掩码预处理（修复/扩绘）

修复训练需要预计算的掩码。

### 仅音频

```bash
python scripts/process_dataset.py     --audio_only     ...
```

## 14.6 输出结构

```
precomputed/
├── video_latents/      # .pt 文件
├── audio_latents/      # .pt 文件（如适用）
├── text_embeddings/    # .pt 文件（video_prompt_embeds 等）
├── reference_latents/  # .pt 文件（IC-LoRA）
├── masks/              # .pt 文件（修复）
└── metadata.json       # 分辨率、帧数等
```
